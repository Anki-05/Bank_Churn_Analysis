# Bank_Churn_Analysis
<img width="605" height="341" alt="image" src="https://github.com/user-attachments/assets/c4f4dab1-d752-42eb-aab3-f9556ac91ca0" />


Overview:
Interactive Power BI dashboard analyzing customer churn in a 10K banking dataset. Overall churn rate: 20.37% (2K exited customers).

Key Insights:
Peak churn: 40-50 age group at 56.04%
Geography hotspot: Germany leads with 32.44% churn
Product effect: 1-product customers churn at 27.71% vs 7.58% for 2 products
Customer profile: 60%+ high-income, 86.5% long-term

Goal: 
Identify high-risk segments and drive targeted retention & cross-selling strategies.

Skills/Tools: Data Cleaning, Modelling, Visualization, Excel, SQL, Power BI
